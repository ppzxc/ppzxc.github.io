Joel On Software, Select Language

2017.04.10

1장에서 물어보는 것은 개발자가 업무를 수행할 때 특정 프로그래밍 언어를 선택하는 이유?인것 같다. 번개같은 속도는 C, 배포판을 윈도우로 한정 짓자면 MFC, C++ 또 맥, 윈도우, 리눅스 처럼 여러 플랫폼에서 구동시킬 경우 Java

작업시간이 촉박하지만 멋진 GUI는 비주얼 베이직 하지만 VB는 배포본의 크기가 커진다. 속력에 무관하고 어떠한 유닉스 운영체제에서도 돌아가는 명령행 도구는 Perl. 웹 브라우저는 JavaScript

요점은 문법, 언어적 특성, systax에만 기준을 두고 언어를 고르는건 우매한 짓이다.

즉, 각각의 언어적 특성을 이해하고 비즈니스에 맞게 또 플랫폼에 맞춰서 고르자.
